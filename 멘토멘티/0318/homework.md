# 연산자
### 연산자 종류
* 산술 연산자 : `+` `-` `*` `/` `%`(나머지)
  
* 증감 연산자 : `++` `--`

* 비교 연산자 : `==` `!=` `>` `<` `>=` `<==`

* 논리 연산자 : `&&`(AND) `!`(NOT) `||`(OR)

* 대입 연산자 : `+=` `-=` `*=` `/=` `%=`

* 삼항 연산자 : `(조건) ? 참_표현식 : 거짓_표현식`

# 조건문

### if문

<pre>if (조건) {
// 실행 코드
}</pre>
조건이 참일때 실행

### else문

<pre>if (조건) {
// 조건이 참일 때 실행되는 코드
} else {
// 만족하는 조건이 없을 때 실행되는 코드
}</pre>
`if`문에서 만족하는 조건이 없을때 `else`문 코드 실행

### else if문

<pre>if (조건1) {
// 작업1 수행
} else if (조건2) {
// 작업2 수행
}</pre>
서로 연관된 조건일때 `if`와 `else if`를 사용 하여 작업 둘 중 하나만 실행되게 한다

### switch문

<pre>switch (조건) {
case value1:
// 조건의 결과 값이 value1일 때 실행되는 코드
break;
case value2:
// 조건의 결과 값이 value2일 때 실행되는 코드
break;
default:
// 조건의 결과 값이 위의 어떤 값에도 해당하지 않을 때 실행되는 코드
   }</pre>
중첩`if`문을 좀 더 편리하게 만든것으로 비교 연산자 사용이 불가 하다(값이 같은지만 확인 가능)

# 반복문
반복할때 사용하는 것으로 모두 중첩 가능하다

### while문
```
while (조건) {
    //조건이 참일때 실행 코드
}
```
조건이 거짓이 될때까지 실행코드를 실행한다

### do while문

```
do {
// 코드
} while (조건식);
```
`while`과 비슷하지만, 조건 참여부와 관계 없이 한번은 꼭 실행한다

### break, continue

```
while(조건식) {
코드1;
break;//즉시 while문 종료로 이동한다.
코드2;
}
//while문 종료

while(조건식) {
코드1;
continue;//즉시 조건식으로 이동한다.
코드2;
}
```


* `break`: 반복문을 즉시 나간다
* `continue`: 나머지 부분을 건너뛰고 다음 반복으로 넘어간다


### for문

```
for (1.초기식; 2.조건식; 4.증감식) {
// 3.코드
}
```
`while`과 비슷한 반목문이고 반복 횟수가 정해져 있을떄 사용한다


# 스코프와 형변환

### 스코프

* `지역 변수`: 특정 지역에서만 사용할 수 있는 변수

* `스코프`: 변수가 접근할 수 있는 범위 <br>
--> `if`문에서 정의된 변수는 안에서만 사용 가능

* 스코프는 변수가 헷갈리게 되는 상황을 막는다

### 형변환

* 작은 범위에서 큰 범위는 담을 수 있다
   * int --> long --> double
* 큰 범위에서 작은 범위는 문제가 발생 한다
    * 소수점 버림
    * 오버플로우
    * 명시적 형변환으로 해결 가능
*  연산과 형변환
    * 같은 것끼린 같은 형을 출력한다
    * 다른것끼린 더 큰 범위로 출력한다
