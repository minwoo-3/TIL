# 게시판에 사용된 기술들

## Spring JPA
- 관계형DB를 사용하는 방법을 정의한 인터페이스

## Spring Boot
- 스프링 프레임워크를 쉽고 빠르게 설정하여 웹 앱을 개발할 수 있게 해주는 도구

## 어노테이션
- 다른 프로그램에게 유용한 정보를 제공하기 위해 사용되는 것으로 주석과 같은 의미
    - @Service, @Column등

## Lombok
- 어노테이션을 기반으로 코드를 자동 완성 시켜주는 라이브러리
    - @Getter, @Setter, @Equlas등

## Spring MVC
- 앱을 개발할 떄 사용하는 디자인 패턴
- Modle, View, Controller로 각각 나눠서 역할에 맞게 코드를 작성하는 방식
