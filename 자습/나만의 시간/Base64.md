# Base64
- 이진 데이터를 64개의 인쇄 가능한 문자로 표현하는 인코딩 방식
  - `A-Z`, `a-z`, `0-9`
  - `+`, `/`
  - `=`

### 인코딩
- 사람이 사용하는 정보를 컴퓨터가 이해할 수 있는 디지털 신호로 변환
- 데이터를 압축해서 전송을 효율적으로 하기 위해 사용
### 디코딩
- 디지털 신호를 사람이 읽을 수 있는 형태로 변환

## 사용되는 상황
- HTML에 작은 이미지나 아이콘 포함
- API 응답에서 이진 데이터 전송
- 이메일 첨부 파일 인코딩
- JWT 토큰

## 사용 이유
- 옛날 시스템들은 순수 이진 데이터를 안정적으로 처리하지 못했음
- 그래서 모두가 공통으로 안전하게 다룰 수 있는 아스키코드 문자만 사용하는 인코딩 방식으로 Base64 등장

## 작동 방식
- 문자를 아스키코드 숫자고 변환
- 2진수로 변환
- 붙혀서 6비트씩 끊기
- 끊은 것을 10진수로 변환
- Base64 인코딩 테이블에서 문자를 찾기

### 예시
```
Man
↓
010011010110000101101110
↓
6비트씩 자름
↓
숫자로 변환
↓
문자로 치환
↓
TWFu
```
