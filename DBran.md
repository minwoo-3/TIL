## 데이터베이스(DB)
- 데이터베이스 : 데이터를 저장하고 조회하는 프로그램

### DB의 데이터 특징
- 통합된 데이터
    - 중복된 정보를 통합하여 용량은 최소화한 데이터의 모임으로 구성
- 저장된 데이터
    - 컴퓨터가 접근할 수 있는 매체에 데이터 저장
- 운영 데이터
    - 조직의 운영을 위해 저장하고 존재하는 운영 데이터
- 공유 데이터
    - 여러사람들이 공유하고 사용할 목적으로 통합 관리되는 데이터
### DB의 기능 특징
- 실시간 접근성
    - 사용자의 요구에 신속하고 정확하게 응답이 가능해야 한다
- 계속적인 변화
    - 현실 세계의 변화 반영을 위해 데이터의 삽입, 삭제, 갱신으로 항상 최신의 데이터를 유지해야 한다
- 동시 공용
    - 다수의 사용자가 동시에 같은 내용의 데이터를 이용할 수 있어야 한다
- 내용에 의한 참조
    - DB의 데이터를 참조할 떄 사용자의 요구에 따른 데이터 내용으로 데이터의 위치나 주소로 데이터를 찾는다
